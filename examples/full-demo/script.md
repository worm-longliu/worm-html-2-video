# 分镜脚本 - HTML Video Creator 技能介绍

## 基本信息
- **总时长：** 52s
- **场景数：** 8 个
- **目标平台：** 抖音/视频号
- **目标观众：** 技术内容创作者、前端开发者

---

## 场景1: 开场Hook（0-4s）
- **画面：** 发光代码图标 + 主标题 "用HTML做抖音视频？"
- **配音：** "用 HTML 就能做抖音视频？2分钟自动出片！"
- **动画：** 图标淡入 → 标题淡入 → 副标题淡入
- **关键元素：** 代码符号 `</>` 发光圆环、主标题、"2分钟出片" 标签

## 场景2: 痛点展示（4-10s）
- **画面：** 传统工具图标对比（AE/PR logo + 学习曲线图）
- **配音：** "传统做法：学 AE 学 PR，光装软件就半天。模板不合适还得从头改，一个 60 秒视频做三天。"
- **动画：** 工具图标依次弹出 → 时间轴展示 → 红色 X 叠加
- **关键元素：** 传统工具列表、时间对比、痛点红色高亮

## 场景3: 解决方案概览（10-17s）
- **画面：** HTML代码 → 视频成品的转换流程图
- **配音：** "现在用 HTML 加 CSS 动画就能做。会写网页就能做视频，帧驱动动画系统，精确到每一帧。"
- **动画：** 代码块淡入 → 箭头流动 → 视频预览弹出
- **关键元素：** HTML代码片段、转换箭头、视频缩略图

## 场景4: 自动化流水线（17-25s）
- **画面：** 四步流程图（Playwright → PNG帧 → Edge-TTS → FFmpeg）
- **配音：** "全程自动化：Playwright 截图 5fps 极速采集，Edge-TTS 免费配音，FFmpeg 一键合成带字幕视频。"
- **动画：** 流程节点依次高亮 → 连线流动 → 最终输出闪烁
- **关键元素：** 四个流程节点、连接线、性能数据标注

## 场景5: 智能字幕（25-32s）
- **画面：** 字幕时间轴可视化 + 算法公式
- **配音：** "字幕精准对齐：语速算法自动计算时间轴，智能换行不拆英文单词，场景边界自动留白。"
- **动画：** 时间轴展开 → 字幕条依次出现 → 公式淡入
- **关键元素：** 时间轴图示、字幕样本、语速公式

## 场景6: 深色主题系统（32-39s）
- **画面：** 颜色分级卡片 + 安全区域示意图
- **配音：** "内置深色主题规范：四级颜色对比度，抖音安全区域适配，视频压缩后依然清晰可读。"
- **动画：** 颜色卡片依次亮起 → 安全区域高亮 → 对比度数值弹出
- **关键元素：** 四色分级、安全区域线、WCAG 对比度值

## 场景7: 多平台适配（39-46s）
- **画面：** 多设备展示（手机竖屏阵列：抖音/视频号/B站/小红书）
- **配音：** "一套 HTML 适配四大平台：抖音竖屏、视频号、B站、小红书。安全区域自动兼容。"
- **动画：** 设备依次弹出 → 平台logo叠加 → 绿色对勾确认
- **关键元素：** 四个手机框、平台标识、兼容对勾

## 场景8: 结尾CTA（46-52s）
- **画面：** 技能Logo + 功能总结 + GitHub地址
- **配音：** "HTML Video Creator，开源免费。关注我，教你从零开始做技术视频！"
- **动画：** Logo脉冲 → 功能标签展开 → GitHub地址弹出
- **关键元素：** 技能名称、功能标签云、开源信息
